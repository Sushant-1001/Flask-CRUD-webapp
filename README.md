This is a simple Flask CRUD (Create, Read, Update, Delete) web application connected to a MySQL database. It allows users to add, view, edit, and delete records from a users table.

The project demonstrates how Flask works with:

Routing
HTML templates (Jinja2)
Form handling
MySQL database operations
Redirects and dynamic URL routing

It is designed as a beginner-friendly full-stack Python project.

# Flask CRUD Application

A simple CRUD (Create, Read, Update, Delete) web application built using **Flask** and **MySQL**.

---

## 🚀 Features

- Add new user data
- View all users
- Edit existing user data
- Delete user records
- MySQL database integration

---

## 🛠️ Tech Stack

- Python (Flask)
- MySQL
- HTML (Jinja2 Templates)
- CSS (Basic styling)
